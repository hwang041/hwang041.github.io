## 概述

伪蒙疆联合自治政府是抗日战争期间日本在中国察哈尔、绥远两省及山西北部扶植的傀儡政权，是日本控制下的殖民工具。其前身为1937年先后成立的三个伪政权——伪蒙古联盟自治政府、伪察南自治政府和伪晋北自治政府。1939年9月1日，三者合并为伪蒙疆联合自治政府，定都张家口。名义上的主席为蒙古王公德穆楚克栋鲁普（德王），实权则掌握在日本最高顾问金井章次及驻蒙军手中。1941年8月对内改称伪"蒙古自治邦"，对外仍用旧名。1945年苏蒙联军进攻、八路军收复张家口后，该政权覆灭。

## 行政区划

伪蒙疆政权的行政区划体现了"蒙汉分治"的特征：蒙古地区沿用盟旗制，汉族聚居区则设省县制。

以1943年为节点，全境分为**五盟、二省、一特别市**：

**蒙古地区（五盟）：**

- **巴彦塔拉盟**（驻厚和，今呼和浩特）——辖土默特旗等5旗及巴彦、萨拉齐、集宁等十余县，是五盟中人口最多、经济最发达的盟。
- **察哈尔盟**（驻张北）——辖察哈尔左翼四旗、商都旗、明安旗等约8旗及张北等5县。
- **锡林郭勒盟**（驻贝子庙，今锡林浩特）——辖乌珠穆沁、浩齐特、苏尼特、阿巴嘎、阿巴哈纳尔五部10旗，地域广袤但人口稀少。
- **乌兰察布盟**（驻百灵庙）——辖四子部落旗、达尔罕旗、茂明安旗、乌拉特三旗等约6旗。
- **伊克昭盟**（驻包头）——名义辖鄂尔多斯七旗，即黄河"几"字弯内的广大区域，但实际大部分仍在重庆国民政府绥远省控制下。

**汉族地区（二省）：**

- **宣化省**（省会宣化）——由察南政厅于1943年1月改设，辖万全、宣化、怀安、怀来、龙关、赤城、延庆、蔚县、涿鹿、阳原10县。
- **大同省**（省会大同）——由晋北政厅于同日改设，辖大同、怀仁、阳高、天镇、浑源、广灵、灵丘、朔平（右玉）、山阴、应县、朔县、左云、平鲁13县。

**首都：**

- **张家口特别市**——直辖于伪政务院，为伪蒙疆政权的政治中心。

## 沿革

| 时间     | 事件                                                       |
| -------- | ---------------------------------------------------------- |
| 1936.5   | 德王在化德成立蒙古军政府                                   |
| 1937.9   | 察南自治政府成立（驻宣化）                                 |
| 1937.10  | 晋北自治政府成立（驻大同）；蒙古联盟自治政府成立（驻归绥） |
| 1937.11  | 三政权组建蒙疆联合委员会（驻张家口）                       |
| 1939.9.1 | 三政权合并为**蒙疆联合自治政府**，定都张家口               |
| 1940     | 汪精卫政权成立，蒙疆名义归属南京，实际独立运作             |
| 1941.8.4 | 对内改称"蒙古自治邦"，改用成吉思汗纪元                     |
| 1943.1.1 | 察南政厅改为宣化省，晋北政厅改为大同省                     |
| 1945.8   | 苏蒙联军攻入，八路军收复张家口，蒙疆政权覆灭               |
